# Quotes App with Firebase Includes:
- Facebook Login 
- Google Login
- Email Login
- List of Thousand of quote
- List of 100+ quote categories
- Copy quotes
- Share quotes
- Add quotes to bookmark.

# Custom Widget 
- Staggered grid view
- Card
- Bubble BottomBar
- Auto resize Text
- Toast
